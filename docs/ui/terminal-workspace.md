## 1. 工作区职责

终端工作区承载多个终端标签：

- 打开终端。
- 关闭终端。
- 断开和重连。
- 显示连接状态。
- 响应窗口尺寸变化。
- 打开当前会话的 SFTP 或端口转发入口。

## 2. 约束

终端控件只在 Presentation 中存在。ViewModel 不持有 SSH 协议库 channel。

终端标签关闭时必须通过 Application 通知关闭会话。
