# AtomSSH 导入导出页

> 文档状态：阶段性冻结
>
> 冻结时间：2026-06-16
>
> 冻结范围：当前文档定义的导入导出页职责和第一版边界
>
> 变更规则：调整导入导出内容、冲突处理或敏感数据策略时必须同步更新本文档。

导入导出页用于 profile、settings、port forward profiles、network inventory、command snippets 等非敏感配置的备份、迁移和恢复。

## 1. 第一版能力

- 导出 profile。
- 导出应用设置。
- 导出端口转发配置。
- 导出网络清单。
- 导出命令片段。
- 导入配置包。
- 导入前预览冲突。
- 冲突处理后提交导入。

## 2. 边界

导入导出页面只通过 Application 用例触发操作，不能直接读写配置文件或仓储。

默认导出不得包含密码、私钥内容、passphrase 或 token。未来如果支持敏感数据导出，必须使用显式加密、二次确认和清楚的风险提示。
