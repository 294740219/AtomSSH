# AtomSSH 传输队列页

> 文档状态：阶段性冻结
>
> 冻结时间：2026-06-16
>
> 冻结范围：当前文档定义的传输队列页职责和交互边界
>
> 变更规则：调整传输队列页面时必须同步更新本文档和 Transfer 模块文档。

传输队列页用于展示等待中和运行中的 SFTP 上传、下载以及远端到远端复制任务。

## 1. 第一版能力

- 展示任务名称、方向、源、目标、状态、进度、速度。
- 取消等待中或运行中的任务。
- 查看失败或中断详情。
- 跳转到传输历史。

## 2. 边界

队列页只通过 Application 查询不可变快照，不能直接访问 Transfer 内部队列、worker、SFTP client 或 SSH channel。

取消任务必须通过 Application 用例触发。
