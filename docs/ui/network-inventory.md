# AtomSSH 网络清单页

> 文档状态：阶段性冻结
>
> 冻结时间：2026-06-16
>
> 冻结范围：当前文档定义的 VPC 内网、跳板机和 SSH 网络清单页职责
>
> 变更规则：调整网络清单 UI 时必须同步更新本文档和 Network 模块文档。

网络清单页用于组织运维网络入口：

- VPC 网段。
- 跳板机。
- 内网目标机器。

## 1. 第一版能力

- 手工添加网络空间。
- 手工添加 VPC 内网机器和跳板机。
- 展示节点地址、标签、环境、在线状态快照。
- 从节点选择或创建 SSH profile，并通过 Application 打开终端。
- 查看连接路径诊断。

## 2. 约束

页面不直接修改云网络配置，不直接创建 SSH client。所有动作通过 Application 用例进入系统。
