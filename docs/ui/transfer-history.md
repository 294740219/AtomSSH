# AtomSSH 传输历史页

>
> 冻结时间：2026-06-16
>
> 冻结范围：当前文档定义的传输历史页职责和交互边界
>
> 变更规则：调整传输历史页面时必须同步更新本文档和 Transfer 模块文档。

传输历史页用于展示已成功、失败、取消和中断的传输任务。

## 1. 第一版能力

- 展示任务摘要、方向、源、目标、结束状态和结束时间。
- 查看失败详情。
- 重试可重试任务。
- 清理历史前显示确认。

## 2. 边界

历史页只展示 Transfer 状态快照，不读取 Transfer worker 或内部队列。重试和清理历史必须通过 Application 用例触发。
