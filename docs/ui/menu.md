## 1. 菜单结构

第一版一级菜单：

- 连接
- 工具
- 设置

连接分组下显示用户 profile 分组和会话节点。

工具：

- SFTP
- 传输队列
- 传输历史
- 端口转发
- 网络清单
- 远端复制
- 命令片段

设置：

- 会话管理
- 应用设置
- 凭据与安全
- 导入导出

## 2. 点击行为

- 点击 profile 节点：通过 Application 打开终端。
- 点击分组：展开或折叠。
- 点击 SFTP：打开当前会话的 SFTP 页面或提示选择会话。
- 点击传输队列：打开运行中和等待中的传输任务。
- 点击传输历史：打开已完成、失败、取消和中断的传输记录。
- 点击端口转发：打开端口转发管理页。
- 点击网络清单：打开 VPC 内网、跳板机和 SSH 节点清单。
- 点击远端复制：打开源主机到目标主机的文件传递工作流。
- 点击会话管理：打开 profile 管理页。
- 点击凭据与安全：打开凭据、Host Key 和 known hosts 管理页。
- 点击导入导出：打开配置备份、迁移和导入冲突处理页。

菜单不直接创建 SSH client。
