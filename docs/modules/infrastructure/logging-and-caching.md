## 1. 日志

日志用于诊断：

- 启动阶段。
- 连接失败。
- 认证失败。
- Host key 拦截。
- SFTP 传输失败。
- 端口转发失败。

日志不得包含 secret material。

## 2. 缓存

第一版仅允许轻量缓存：

- 最近打开 profile。
- UI 展开状态。
- 最近目录路径。

缓存不能成为业务事实来源。profile、known hosts、传输任务必须由正式仓储保存。
